Suggested 14-Week Timeline

Week Topics
1Fundamental concepts of computing, history of computing, natural vs programming languages, programming lifecycle, compilers, OS, memory, and I/O
2Python expressions, Python shell, Python as a calculator, variables, syntax and runtime errors, sequence of execution, comments
3Data types and objects, concept of functions, print and input, advanced expressions, strings
4Simple built-in functions, lists, list operations and methods, use of map
5Boolean values, relational and logical operators, membership operators, conditional statements
6Using the random library, comparing strings and lists, basic problem solving with conditionals
7Practice problems with Booleans, lists, and conditionals
8Repetition statements (loops), loop components (start, progress, end), while loops
9Flowcharts, infinite loops, sentinel values, simple loop problems (sum, length, max, min)
10Steps to solve a problem, break and continue, more examples, introduction to for loops
11Different forms of for loops, comparison with while loops, basic numerical problems (e.g., prime testing)
12Defining and calling functions, introduction to file processing: opening, reading, writing, and closing files
13Sets and dictionaries, basic file-based problem solving
14Comprehensive review and data analysis problems using loops, files, and functions
