Course Overview

  • Prerequisites: Basic understanding of computers, high-school diploma or two-year college diploma, passing placement test.
  • Tools Requirement: Networked computer lab with Windows and Linux systems.

Course Schedule

Topic Day Form
Computation, computers, computer hardware, software, systems, and the web.1Conceptual
Operating system concepts, servers, Linux, and UNIX.2Conceptual
Working with Windows and Linux administrative tools, users, permissions, groups, and command line tools.3, 4Practical
Introduction to computer networks, network protocols (IP), interfaces, data transmission.5Conceptual
Hands-on networking: Secure shell (SSH), Telnet, networking requests in browsers, netstat, nslookup, ifconfig, dig.6, 7Practical
Cybersecurity concepts: Threats, security principles, taxonomy of attacks (malware, phishing, DoS, cryptographic attacks, IoT, mobile threats).8, 9Conceptual
Cryptography: Encryption, decryption, ciphers, OpenSSL basics, digital signatures, encryption in applications.10Hybrid

Course Outcomes

  • Understand basic concepts of computers, software, networks, and security.
  • Use open-source tools in Windows and Linux for network analysis and cryptography.
