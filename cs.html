Course Overview

This course provides a tour of computer systems fundamentals, providing an understanding of the relationship between hardware and software, basic data representation in memory, the x86 instruction set, basic processor design, and program optimization.

This course is taught by Dr. Hussain Almohri at the Computer Science department, Kuwait University.

Required textbook: Computer Systems: A Programmer's Perspective (3rd Edition)

Tools and platform: We will use the GCC and the Ubuntu Linux as our testing environment.
